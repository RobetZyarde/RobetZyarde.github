一些想说的 

努力对自己说清楚

心廃棄少年一次一次的围绕跑道毫无目的的旋转，
旋转在自己的世界里，迷茫不知所措。
以为这种郁闷独此一份（呵呵），可发现原来身边就有一样的人，
我还曾经看过那些话，‘一些想说的’，粗粗一扫，什么都没有留下，
只是心里空虚四处觅食，
根本没有放在心上，
然后，然后，
百无聊赖，
又读了一遍，

是这样的故事啊，
女孩子也会这样啊，
（还是只有女孩子会这样 这..）
曾经也有这样一个故事…
长长的信永远躺在草稿箱，我根本没有表白，
她却越来越频繁的问候大学的日常，
发来的明信片写些可有可无的感谢的话，
我无法接受，我又生气了，
不理她的一切，草草回答，
我的朋友本来就很少，
她也是那么普通，长相很一般，
家庭也很呆板，没什么思想，
为什么我曾经这么喜欢她，
为什么我曾经和她聊天都那么高兴，
为了她失去了许多（根本没用的）机会，
我无情的删除了那个她，憎恨的名单上又莫名多了一个人，
我活在《为情所困？》 《爱情不是生存必要，而是锦上添花》的阴影下，
活在“宁缺毋滥”的压力下，
我就算是“滥”吧，连唯一的“人品好”这个优点也彻底败掉了。

然后反反复复，再次验证了，我的确是烂。
“有大神教我了”
“我一知道这么可爱的孩子是生医的就想到了你，可他也是大神”
“二姐很厉害了，不用和他比”
…
“他有什么厉害的”
..
我曾经试着接受这一切，
假装和颜悦色，
有一瞬这个念头，”哎，当个备胎也不错，反正这两个傻逼好不了多久”
放他妈的血，
我怎么可能接受这种设定，
也许我最初就没有所谓的”真诚”
连备胎都不敢做，还说什么真心！
果然他们好不了多久就散了，她也果然选择了备胎，
她喜欢“爱”，也有实力永远拥有“爱”，
对她来说这一场收获很多，对我来说什么也不是，
而我早已把她的微信、QQ全部删光，也早早离开了支教，
我无法忍受，无法改变，精神错乱，
更加害怕伤害，变得不愿意付出，
耻辱，
可我很现实，我根本无法安慰自己，
但我这样的状态，短期内，实在找不到”更好的”（长期也难说）
所以我又开始渴望离开这里，
离开这，留下那个不完整的我，

连漫画里都写“年轻人都是欲望的动物”
我更是热衷于用行动解决问题，
要去，努力，却毫无，动力，
啊啊啊啊啊啊啊啊（跺脚、跺脚），
这根本不是想清楚就能解决的问题，
我也是没把感情用在对社会有用的地方。

荒唐的喜剧。
我却渴望说出来，
我却渴望倾听。
作为以科学为信仰的人，其实我鄙视有”信仰的人”，
可福音可以让你收获一群敞开心扉的朋友，
对于弱小的人来说，也不错。

arm单片机起步（二）

时钟与定时器

ARM的时钟与其相关的总线比C51复杂了很多。

在使用库函数的时候，有专门的函数用来初始化。

通过使用相应的函数，可以减少对其相关寄存器的详细了解。

从此图可以看出，每个具体外设的时钟总线是固定的，在设置时可以查阅相应的文档。

系统时钟的初始化

在system_stm32f10x.c文件中有设计时钟的宏定义，最终在SystemInit()中被调用。

一般情况下是不用修改推荐的定义值的。

文件开头就有一个这样的定义:

//#define SYSCLK_FREQ_HSE    HSE_Value
//#define SYSCLK_FREQ_20MHz 20000000
//#define SYSCLK_FREQ_36MHz 36000000
//#define SYSCLK_FREQ_48MHz 48000000
//#define SYSCLK_FREQ_56MHz 56000000
  #define SYSCLK_FREQ_72MHz 72000000

通过 #define 和 #elif defined 的组合使用，让用户从软件上自由选择系统时钟，

而厂家提供推荐的各个总线下的时钟分频频率（见c文件中相关定义）

定时器时钟的初始化

以定时器举例
TIM_TimeBaseInitTypeDef TIM_TimeBaseStructure;
NVIC_InitTypeDef NVIC_InitStructure;

RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM4, ENABLE); //时钟使能

//定时器TIM3初始化
TIM_TimeBaseStructure.TIM_Period = 1000-1; //设置在下一个更新事件装入活动的自动重装载寄存器周期的值
TIM_TimeBaseStructure.TIM_Prescaler = 72-1; //设置用来作为TIMx时钟频率除数的预分频值
TIM_TimeBaseStructure.TIM_ClockDivision = 0;
TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;  //TIM向上计数模式
TIM_TimeBaseInit(TIM4, &TIM_TimeBaseStructure); //根据指定的参数初始化TIMx的时间基数单位

TIM_ITConfig(TIM4,TIM_IT_Update,ENABLE ); //使能指定的TIM3中断,允许更新中断

//中断优先级NVIC设置
NVIC_InitStructure.NVIC_IRQChannel = TIM4_IRQn;  //TIM3中断
NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 2;  //先占优先级0级
NVIC_InitStructure.NVIC_IRQChannelSubPriority = 3;  //从优先级3级
NVIC_InitStructure.NVIC_IRQChannelCmd = ENABLE; //IRQ通道被使能
NVIC_Init(&NVIC_InitStructure);  //初始化NVIC寄存器

TIM_Cmd(TIM4, ENABLE);  //使能TIMx

此设置的目的为提供一个周期为1ms的中断。

由上面的设置我们不难看出APB1总线的频率为72M。

这里我设置的分频数为72，则计数器的频率为1M。

Period本质上是设置的比较器的值，设置位1000后，1M/1000 = 1000Hz

得到了1ms的周期。

到目前为止也就两次，

似乎“被我搞糟了”，

也许生命不该是这个样子，

But who care?

我还是要跪在你们面前，

戴着面具装小狗，

活着就求有一天，

好好教你们这些无耻的人做人的道理。

我曾乞求岁月匆匆
带走你明亮的双眼
你如桃花般娇嫩的皮肤
和你迷人的青春朝气
那样我就可以独自爱你
你也会在乎我的爱
岁月真的匆匆过了
带走了你明亮的眼睛
你如桃花般娇嫩的皮肤
和你迷人的青春朝气
可是我却不再爱你
也不再在乎你的爱
